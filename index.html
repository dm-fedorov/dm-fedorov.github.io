Крутые проекты Python для разработчиков игр

- Камень, ножницы, бумага (https://thehelloworldprogram.com/python/python-game-rock-paper-scissors/). Начните свое путешествие по изучению Python с простой, но увлекательной игры, которую знают все.

- Создание текстовой приключенческой игры (https://coding-grace-guide.readthedocs.io/en/latest/guide/lessonplans/beginners-python-text-based-adventure.html) — это классический проект Python для начинающих, который научит вас многим основным концепциям настройки игры, которые будут полезны для более продвинутых игр.

- Игра в угадайку (https://djangocentral.com/creating-a-guessing-game-in-python/) — это еще один проект для начинающих, который поможет вам изучить и попрактиковаться в основах.

- Mad Libs (https://www.mikedane.com/programming-languages/python/building-a-mad-libs-game/) — узнайте, как сделать интерактивную Python Mad Libs!

- Hangman (https://selftaught.blog/python-tutorial-build-hangman/) — еще одна классика детства, которую вы можете сделать на Python, чтобы развить свои навыки.

- Змейка (https://www.edureka.co/blog/snake-game-with-pygame/). Это немного сложнее, но это классическая (и на удивление забавная) игра, в которую можно играть и играть.

Идеи проекта Python: начальный уровень

- Создайте генератор кода. Это может принимать текст в качестве входных данных, заменять каждую букву другой буквой и выводить «закодированное» сообщение.

- Создайте калькулятор обратного отсчета. Напишите код, который может принимать две даты в качестве входных данных, а затем вычислять время между ними. Это отличный способ познакомиться с модулем даты и времени Python.

- Напишите метод сортировки. Учитывая список, можете ли вы написать код, который сортирует его в алфавитном или числовом порядке? Да, Python имеет встроенную функциональность, но посмотрите, сможете ли вы сделать это без использования sort()!

- Создайте интерактивную викторину. Какой ты Мститель? Создайте викторину о личности или рекомендации, которая задает пользователям несколько вопросов, сохраняет их ответы, а затем выполняет какие-то вычисления, чтобы предоставить пользователю персонализированный результат на основе их ответов.

- Крестики-нолики по тексту. Создайте игру «Крестики-нолики», в которую можно играть как в текстовое приключение. Можете ли вы заставить его печатать текстовое представление доски после каждого хода?

- Сделайте преобразователь температуры/измерения. Напишите сценарий, который может конвертировать градусы Фаренгейта в градусы Цельсия и обратно, дюймы в сантиметры и обратно и т. д. Как далеко вы сможете зайти?

- Создайте приложение счетчика. Сделайте свои первые шаги в мире пользовательского интерфейса, создав очень простое приложение, которое будет увеличиваться на единицу каждый раз, когда пользователь нажимает кнопку.

- Создайте игру на угадывание чисел. Думайте об этом как о текстовом приключении, но с цифрами. Как далеко вы можете зайти?

- Построить будильник. Это пограничный начальный/средний уровень, но стоит попробовать сделать будильник для себя. Можно ли создавать разные будильники? Функция повтора?

- Генератор паролей (https://www.101computing.net/random-password-generator/) — создайте безопасный генератор паролей на Python.

- Создайте адресную книгу. Это может начаться с простого словаря Python или стать чем-то вроде этого (https://code-projects.org/address-book-in-python-with-source-code/)

- Кодировщик/декодер QR-кода Python Project. В этом уроке Code With Tomi (https://www.youtube.com/watch?v=SqvVm3QiQVk&t=3192s) вы узнаете, как создавать свои собственные QR-коды и кодировать/декодировать информацию из них. В этом проекте используется qrcodeбиблиотека.

- Крестики-нолики Python Project. В этом уроке Кайли Ин (https://www.youtube.com/watch?v=8ext9G7xspg&t=2153s) вы узнаете, как создать игру в крестики-нолики с разными игроками в командной строке. Вы узнаете, как работать с Python timeи mathмодулями, а также получите постоянную практику с вложенными операторами if.

- Крестики-нолики AI Python Project. В этом уроке Кайли Ин (https://www.youtube.com/watch?v=8ext9G7xspg&t=3599s) вы узнаете, как создать игру в крестики-нолики, в которой компьютер никогда не проигрывает. В этом проекте используется минимаксный алгоритм, который является рекурсивным алгоритмом, используемым для принятия решений.

- Проект Python для двоичного поиска. В этом руководстве Кайли Ин (https://www.youtube.com/watch?v=8ext9G7xspg&t=4553s) вы узнаете, как реализовать алгоритм «разделяй и властвуй», называемый бинарным поиском. Это распространенный алгоритм поиска, который встречается на собеседованиях, поэтому важно знать, как реализовать его в коде.

- Обработка фотографий в проекте Python. В этом уроке Кайли Ин (https://www.youtube.com/watch?v=8ext9G7xspg&t=7534s) вы узнаете, как создать фильтр изображения и изменить контрастность, яркость и размытие изображений. Перед запуском проекта вам необходимо скачать стартовые файлы (https://github.com/kying18/pyphotoshop).

- Марковская цепочка Text Composer Python Project. В этом уроке Кайли Ин (https://www.youtube.com/watch?v=8ext9G7xspg&t=9109s) вы узнаете о графовой модели цепи Маркова и о том, как ее можно применить к взаимосвязи текстов песен. Этот проект — отличное введение в искусственный интеллект в Python.

- Проект Pong Python. В этом уроке Кристиана Томпсона (https://www.youtube.com/watch?v=XGf2GcyHPhc&t=78s) вы узнаете, как воссоздать классическую игру в понг на Python. Вы будете работать с модулями osи turtlePython, которые отлично подходят для создания графики для игр.

- Змеиный проект Python. В этом уроке Tech with Tim (https://www.youtube.com/watch?v=XGf2GcyHPhc&t=2736s) вы узнаете, как воссоздать классическую игру про змей на Python. В этом проекте используется объектно-ориентированное программирование и Pygame — популярный модуль Python для создания игр.

- Соедините четыре проекта Python. В этом уроке от Кейта Галли (https://www.youtube.com/watch?v=XGf2GcyHPhc&t=5697s) вы узнаете, как создать классическую игру «Соедини четыре». В этом проекте используются модули numpy, и Python.mathpygamesys

Идеи проекта Python: средний уровень

- Создайте обновленный генератор кода. Начав с проекта, упомянутого в разделе для начинающих, посмотрите, что вы можете сделать, чтобы сделать его более сложным. Можете ли вы заставить его генерировать различные виды кодов? Можете ли вы создать приложение «декодер», которое читает закодированные сообщения, если пользователь вводит секретный ключ? Можете ли вы создать более сложный код, выходящий за рамки простой замены букв?

- Сделайте свою игру в крестики-нолики кликабельной. На основе проекта для начинающих создайте версию игры «Крестики-нолики» с реальным пользовательским интерфейсом, который вы будете использовать, нажимая на открытые квадраты. Задача: можете ли вы написать простого противника «ИИ», против которого может играть игрок-человек?

- Соберите некоторые данные для анализа. Это действительно может быть что угодно, с любого веб-сайта, который вам нравится. Интернет полон интересных данных, и если вы немного узнаете о веб-скрейпинге, вы сможете собрать действительно уникальные наборы данных.

- Автоматизируйте часть своей работы. Это может быть разным, но во многих работах есть повторяющийся процесс, который вы можете автоматизировать!

- Автоматизируйте свои личные привычки. Вы хотите не забывать вставать каждый час во время работы? Как насчет написания кода, который генерирует уникальные планы тренировок на основе ваших целей и предпочтений? Существует множество простых приложений, которые вы можете создать для себя, чтобы автоматизировать или улучшить различные аспекты своей жизни.

- Создайте простой веб-браузер. Создайте простой пользовательский интерфейс, который принимает URL-адреса и загружает веб-страницы. PyWt будет полезен здесь! Можете ли вы добавить кнопку «назад», закладки и другие интересные функции?

- Напишите приложение для заметок. Создайте приложение, которое поможет людям писать и хранить заметки. Можете ли вы придумать какие-нибудь интересные и уникальные функции, которые можно добавить?

- Создайте тестер набора текста. Это должно показывать пользователю некоторый текст, а затем побуждать его набирать его — при этом синхронизируя их и оценивая их точность.

- Создайте систему уведомлений об обновлении сайта. Вы когда-нибудь раздражались, когда вам приходилось обновлять веб-сайт, чтобы увидеть, был ли повторно выставлен товар, которого нет в наличии, или чтобы увидеть, были ли опубликованы какие-либо новости? Напишите скрипт Python, который автоматически проверяет заданный URL-адрес на наличие обновлений и информирует вас об их обнаружении. (Однако будьте осторожны, чтобы не перегрузить серверы любого сайта, который вы проверяете — соблюдайте разумный интервал времени между каждой проверкой).

- Воссоздайте свою любимую настольную игру на Python. Здесь есть множество вариантов, от чего-то простого, такого как Checkers, до Risk — или даже более современных и продвинутых игр, таких как Ticket to Ride или Settlers of Catan. Насколько близко вы можете подобраться к настоящему?

- Создайте обозреватель Википедии. Создайте приложение, которое отображает случайную страницу Википедии. Проблема здесь в деталях: можете ли вы добавить выбранные пользователем категории? Можете ли вы попробовать другую версию приложения «кроличья нора», в которой каждая статья выбирается случайным образом из статей, на которые есть ссылки в предыдущей статье? Это может показаться простым, но на самом деле это может потребовать серьезных навыков веб-скрейпинга.

- Тетрис Python Project. В этом уроке «Технологии с Тимом» (https://www.youtube.com/watch?v=XGf2GcyHPhc&t=9756s) вы узнаете, как воссоздать классическую игру «Тетрис». Этот проект использует Pygame и отлично подходит для начинающих разработчиков, чтобы поднять свои навыки на новый уровень.

- Многопользовательская онлайн-игра Python Project. В этом руководстве по технологии с Тимом (https://www.youtube.com/watch?v=XGf2GcyHPhc&t=15732s) вы узнаете, как создать многопользовательскую онлайн-игру, в которую вы сможете играть с кем угодно по всему миру. Этот проект — отличное введение в работу с сокетами, сетью и Pygame.

- Программа веб-скрейпинга Python Project. В этом руководстве по Code With Tomi (https://www.youtube.com/watch?v=SqvVm3QiQVk&t=37s) вы узнаете, как запрашивать пользовательский ввод для ссылки пользователя GitHub и выводить ссылку на изображение профиля с помощью веб-скрейпинга. Веб-скрапинг — это метод, который собирает данные с веб-страницы.

- Массовое переименование файлов Python Project. В этом руководстве Code With Tomi (https://www.youtube.com/watch?v=SqvVm3QiQVk&t=833s) вы узнаете, как создать программу, которая может открывать любую папку на вашем компьютере и переименовывать все файлы в соответствии с условиями, установленными в вашем коде Python.

- Погодная программа Python Project. В этом учебном пособии Code With Tomi (https://www.youtube.com/watch?v=SqvVm3QiQVk&t=1494s) вы узнаете, как создать программу, которая собирает пользовательские данные в определенном месте и выводит сведения о погоде в указанном месте. Это отличный проект, чтобы начать изучать, как получать данные из API.

- Космические захватчики Game Python Project. В этом руководстве по сборке с Python (https://www.youtube.com/watch?v=FfWpgLFMI7w) вы узнаете, как создать игру космических захватчиков с помощью Pygame. Вы узнаете множество основ разработки игр, таких как игровые циклы, обнаружение столкновений, события нажатия клавиш и многое другое.
